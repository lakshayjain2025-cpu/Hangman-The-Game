  Hangman Game made using Python

Players can guess words from multiple categories, use hints, and enjoy a clean interface.

* Features
- Multiple word categories (animals, countries, etc.)
- Optional hint system
- Sound effects for correct/wrong answers and win/loss
- Statistics tracking (wins, losses, accuracy)
- Simple and colorful GUI built with Tkinter

 Installation

1. Clone this repository
  
   git clone https://github.com/lakshayjain2025-cpu/Hangman-The-Game.git
   cd Hangman-The-Game
   pip install -r requirements.txt
   python HangmanGame.py
